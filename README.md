# Ultimate Spinner

A customizable, animated spinner component for SvelteKit applications.

## Installation

```bash
npm install ultimate-spinner
# or
pnpm add ultimate-spinner
```

## Usage

```svelte
<script>
  import { Spinner } from 'ultimate-spinner';
</script>

<Spinner 
  items={['Item 1', 'Item 2', 'Item 3']}
  spinDuration={3000}
  title="My Spinner"
/>
```
